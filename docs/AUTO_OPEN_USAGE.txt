================================================================================
                         自动打开功能 - 快速使用指南
================================================================================

🎉 新功能：生成HTML后自动在浏览器中打开！

--------------------------------------------------------------------------------
使用方法
--------------------------------------------------------------------------------

【新版程序】
  运行: python main.py

  在配置阶段，会看到：

    生成后自动打开文件？
    → 是（推荐）- 自动在浏览器中打开
      否 - 仅保存文件

  使用方向键选择，回车确认。

【旧版程序】
  运行: python waveform_viewer.py

  默认自动打开，无需配置。

--------------------------------------------------------------------------------
效果
--------------------------------------------------------------------------------

  选择"是"：
    ✓ 可视化结果已保存到: waveform_visualization.html
    ✓ 已在浏览器中打开        <-- 自动在默认浏览器打开

  选择"否"：
    ✓ 可视化结果已保存到: waveform_visualization.html
    （不会自动打开）

--------------------------------------------------------------------------------
适用场景
--------------------------------------------------------------------------------

  ✓ 适合自动打开：
    • 手动运行程序查看单个文件
    • 快速检查可视化结果
    • 演示和展示

  ✗ 不适合自动打开：
    • 批量处理多个文件（会打开很多标签）
    • 自动化脚本（无人值守）
    • 服务器环境（无图形界面）

--------------------------------------------------------------------------------
编程使用
--------------------------------------------------------------------------------

  from waveform_viewer.visualizers.base import VisualizationConfig

  # 启用自动打开
  config = VisualizationConfig(auto_open=True)

  # 禁用自动打开
  config = VisualizationConfig(auto_open=False)

--------------------------------------------------------------------------------
常见问题
--------------------------------------------------------------------------------

  Q: 没有自动打开？
  A: 检查是否选择了"是"，以及系统是否有默认浏览器。

  Q: 打开了错误的浏览器？
  A: webbrowser使用系统默认浏览器，请修改系统设置。

  Q: 批量处理打开太多标签？
  A: 选择"否 - 仅保存文件"即可。
